# Installation overview

## What is it?

This section covers how to install and configure the Java Agent on a Windows system so that OpCon can communicate with it and schedule jobs.

- Use the new installation procedure when setting up the Java Agent on a machine for the first time
- Use the upgrade procedure when updating an existing Java Agent to a newer release
- Use the silent mode reference when automating installations without user interaction
