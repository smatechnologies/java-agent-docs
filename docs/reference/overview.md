# Reference overview

## What is it?

This section contains reference material for the Java Agent, including all messages the agent produces during startup and job processing. Use it to look up specific messages when reviewing job output or troubleshooting agent behavior.

- Use the machine messages reference when a job fails and you need to understand the error message in the job output log
- Use the machine messages reference when reviewing the `JavaAgent.log` file to diagnose startup or connectivity issues
